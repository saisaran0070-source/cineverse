# Contributing to Cineverse

Thank you for considering contributing to the Cineverse project! We appreciate your help in enhancing our community and improving the software.

## Code Style
- Follow the coding conventions used in the project.
- Use meaningful variable names.
- Keep your code clean and readable.
- Write comments where necessary to explain complex logic.

## Pull Request Process
1. Fork the project and create your branch from `main`.
2. Make your changes and ensure they are working properly.
3. Commit your changes with a clear message.
4. Push to your forked branch.
5. Open a pull request to the `main` branch of the original repository.
6. Ensure your pull request references any relevant issues.

## Issue Templates
- Please use the appropriate issue template for bug reports or feature requests.
- Provide as much detail as possible when describing issues or suggesting features.
- Include steps to reproduce any bugs if applicable.

Thank you for your contributions! Together we can make Cineverse better!
